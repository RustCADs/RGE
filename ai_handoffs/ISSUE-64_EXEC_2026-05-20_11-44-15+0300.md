### Tests

- `crates/editor-ui/tests/phase8_graph_diff_smoke.rs::graph_diff_between_reports_one_added_node_and_edge_for_all_three_phase8_domains`:
  the single new integration test target for this dispatch. It defines a
  test-local `DiffNode { display_name: String, kind: String }` payload
  (deriving `Clone`, `Debug`, `PartialEq`, `Eq`) and a test-local helper
  `snapshot_via_adapter(&dyn VizAdapter) -> GraphSnapshot<DiffNode, String>`
  that populates a temporary `Graph<DiffNode, String>` from `adapter.nodes()`
  + `adapter.edges()`, preserving each adapter `NodeId`, `EdgeId`, edge
  `src`, edge `dst`, node display name, node kind, and edge label exactly.

  The test then exercises three domains with the same helper and the same
  `GraphDiff::between(&old_snapshot, &new_snapshot)` call shape:

  - **Material domain (`assert_material_domain`)** — old: `albedo -> output`
    with `MaterialEdge { src_port: Color, dst_port: Color }`. New: also
    adds a `normal` node and a `normal -> output` edge with
    `MaterialEdge { src_port: Vector, dst_port: Texture }`. Asserts the
    diff reports exactly the `normal` node with projected payload
    `DiffNode { display_name: "normal", kind: "MaterialNode" }` and the
    edge record `EdgeRecord { src: normal, dst: output, data:
    "vector->texture" }`.
  - **Animation domain (`assert_anim_domain`)** — old: `idle -> run` via
    `AnimTransition::new("start_run")`. New: also adds a `jump` state and
    a `run -> jump` transition via `AnimTransition::new("leap")`. Asserts
    the diff reports exactly the `jump` node with projected payload
    `DiffNode { display_name: "jump", kind: "AnimState" }` and the edge
    record `EdgeRecord { src: run, dst: jump, data: "leap" }`.
  - **Operator domain (`assert_operator_domain`)** — old: `Cuboid ->
    first_transform` on input port `0`. New: also adds a second `Transform`
    node (distinct translation payload `[2.0, 0.0, 0.0]` vs. the first
    `[1.0, 0.0, 0.0]` so the content-derived `NodeId`s do not collide) and
    a `first_transform -> second_transform` edge on input port `0`. Asserts
    the diff reports exactly the second `Transform` node with projected
    payload `DiffNode { display_name: "Transform", kind: "Transform" }`
    and the edge record `EdgeRecord { src: first_transform, dst:
    second_transform, data: "input[0]" }`.

  Each domain block also asserts `removed_nodes`, `removed_edges`,
  `changed_nodes`, and `changed_edges` are empty, that
  `diff.node_change_count() == 1`, `diff.edge_change_count() == 1`, and
  `!diff.is_empty()`.
