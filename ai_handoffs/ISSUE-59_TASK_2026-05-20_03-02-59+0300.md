- Add exactly one new unit test named `invalidation_propagates_through_operator_outgoing_edges` in the existing `#[cfg(test)] mod tests` block of `crates/cad-core/src/graph/operator_graph.rs`.
- The test must build a diamond-shaped `OperatorGraph` with one upstream operator feeding two branch operators, both of which feed one shared convergence operator.
