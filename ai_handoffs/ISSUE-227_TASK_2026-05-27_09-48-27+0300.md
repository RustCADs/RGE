## Deliverables

- A small pure helper in `Invoke-AiDispatchQueue.ps1` that accepts the already
  computed queue state: `$willRetry`, `$runFailed`, `$taxonomyLabels`,
  `$QueueLabel`, `$runLabel`, `$doneLabel`, `$failLabel`, `$retryLabel`, and
  the known failure-taxonomy label set, and returns the intended terminal
  label add/remove plan.
- The final terminal `gh issue edit` label mutation uses that helper's plan.
- Existing post-mutation verification is preserved and strengthened as needed
  to verify incompatible labels are absent, not only that expected labels are
  present.
- Focused Pester coverage under `tools/dispatch-tests/**` for the pure helper.
  Tests must not call `gh`, create real issues, run the queue, run Codex or
  Claude, invoke the scheduler, or touch the network.
- `AI_DISPATCH_AUTOMATION.md` is updated only if needed to document the final
  terminal label-state convention.

## Acceptance Criteria

- Terminal success (`$runFailed -eq $false`, `$willRetry -eq $false`) adds
  `ai-dispatch-done` and removes `ai-dispatch`, `ai-dispatch-running`,
  `ai-dispatch-retry`, `ai-dispatch-failed`, and every
  `ai-dispatch-failure-*` taxonomy label.
- Terminal failure (`$runFailed -eq $true`, `$willRetry -eq $false`) adds
  `ai-dispatch-done`, `ai-dispatch-failed`, and the selected taxonomy labels,
  and removes `ai-dispatch`, `ai-dispatch-running`, `ai-dispatch-retry`, and
  every non-selected `ai-dispatch-failure-*` taxonomy label.
- Retry (`$willRetry -eq $true`) keeps or adds `ai-dispatch` and
  `ai-dispatch-retry`, and removes `ai-dispatch-running`, `ai-dispatch-done`,
  `ai-dispatch-failed`, and every `ai-dispatch-failure-*` taxonomy label.
- The label-plan helper is deterministic, side-effect-free, and covered by
  Pester tests for success, terminal failure, retry, and stale/incompatible
  taxonomy label removal.
- No test performs live GitHub calls, creates real issues, runs the queue, runs
  Codex or Claude, invokes the scheduler, or touches the network.
- Static inspection confirms no changes outside the allowed file surface and
  this dispatch's generated artifacts.
